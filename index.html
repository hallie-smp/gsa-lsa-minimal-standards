⚙️ Supabase Setup Required

To save and view client history, connect a free Supabase database:

  1. Create a free account at supabase.com → New project
  2. Go to SQL Editor and run the setup script below
  3. Go to Settings → API and copy your Project URL and anon key
  4. Open index.html and replace SUPABASE_URL and SUPABASE_ANON_KEY at the top of the script

SQL to run in Supabase:

create table metric_entries (
  id uuid default gen_random_uuid() primary key,
  created_at timestamptz default now(),
  client_name text not null,
  report_month text not null,
  platform text not null,
  shop_type text not null,
  service_area text not null,
  metric text not null,
  value numeric not null,
  status text not null,
  reliability text
);
-- Allow all team members to read/write (adjust if you want row-level security)
alter table metric_entries enable row level security;
create policy "allow_all" on metric_entries for all using (true) with check (true);
